# System Usability Scale (SUS)

Para cada afirmação, pedir que assinale de 1 (Discordo fortemente) a 5 (Concordo fortemente).

1. Eu penso que gostaria de usar este sistema com frequência.
2. Achei o sistema desnecessariamente complexo.
3. Achei o sistema fácil de usar.
4. Acho que preciso de apoio técnico para usar o sistema.
5. Achei as várias funções bem integradas.
6. Achei que houve muita inconsistência.
7. Imagino que a maioria das pessoas aprenderia rapidamente a usar este sistema.
8. Achei o sistema difícil de usar.
9. Senti-me confiante a usar o sistema.
10. Preciso de aprender muitas coisas antes de usar o sistema.

Pontuação: calcular SUS padrão (0–100). Interpretação: <50 pobre, 50–70 aceitável, >70 bom.
