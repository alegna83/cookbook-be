# Recrutamento — Conteúdo do Formulário (transferir para Google Forms)

Campos sugeridos:

1. Nome completo (texto)
2. Email (texto)
3. Idade (faixas)
4. Dispositivo principal que vais usar (Android / iOS / Desktop - browser)
5. Experiência com Peregrinações (Nenhuma / Já fiz o Caminho / Outro)
6. Já usaste apps de alojamento ou UGC antes? (Sim/Não) — qual?
7. Disponibilidade para realizar o teste (datas/semana seguinte)
8. Aceito participar e concordo com o termo de consentimento (checkbox) — link para `consent.md`
9. Aceito que a sessão (áudio/ecrã) seja gravada e usada para análise (checkbox)
10. Preferes ser contactado por email ou WhatsApp? (campo livre)

Instruções finais no formulário (texto):
- Depois de completares as tarefas conforme as instruções que iremos enviar, por favor preenche as métricas pedidas (sucesso, tempo, SUS) no mesmo formulário ou envia por email.
